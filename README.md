## Run examples
Prerequisites: Install typescript and ts-node

```
npm install -g ts-node
npm install -g typescript
```

To run a typescript example
```
ts-node QuickFindAlg.ts
```

To run java samples

```
javac QuickFindAlg.java && java QuickFindAlg
```
